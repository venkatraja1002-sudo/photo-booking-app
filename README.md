# photo-booking-app
Photography Booking &amp; Portfolio Web App with Admin Panel, WhatsApp Integration, and Package Management built using Streamlit &amp; SQLite.

# 📸 EAGLE DIGITAL AND GRAPHICS — Photography Booking App

A Streamlit-based Photography Booking Web App where customers can view packages and submit booking requests.  
Includes an Admin Panel to manage bookings and update package details (CRUD). Also includes WhatsApp integration for quick communication.

---

## 🚀 Live Demo
👉 Paste your Streamlit Cloud link here:  
**Demo:** https://YOUR-STREAMLIT-LINK

---

## ✨ Features

### Customer Side
- View photography packages
- Book an event (name, phone, date, location, package, message)
- WhatsApp “Message Us” link after booking submission
- Luxury UI contact buttons (WhatsApp / Call / Email)

### Admin Side
- Secure Admin login using Streamlit Secrets
- View all bookings in a table
- Update booking status (pending / confirmed / rejected)
- Manage Packages (CRUD)
  - Add new package
  - Edit package details
  - Activate / Deactivate packages

---

## 🛠 Tech Stack
- **Frontend:** Streamlit
- **Backend:** Python
- **Database:** SQLite
- **Deployment:** Streamlit Cloud
